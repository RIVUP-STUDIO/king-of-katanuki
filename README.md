# PROJECT ENNICHI ─ KING OF KATANUKI（第一弾：縁日）

昔ながらの縁日「型抜き」を再現する、超シンプル集中型のスマホゲームです。

ストーリーもキャラクターもボス戦もありません。
ルールはひとつ。線からズレず、なぞりきる。それだけです。

第一弾のテーマは「縁日」。夜店・提灯の明かり・飴・和紙・夏祭りの、
少し懐かしい世界観で10ステージ収録しています。

## 遊び方

1. ステージを選ぶ（全10ステージ、下記参照）
2. 画面上を指でなぞる（指先そのものではなく、指から少し離れた場所に「針先」が表示され、そこが判定点になります）
3. 針先が線の上に乗ると **緑**（セーフ）
4. 線から外側にズレても **黄色**（まだセーフ、戻せば続行できます）
5. 線の内側へ浅く入っただけなら **黄色**（警告）、深く入ると **赤**（即ゲームオーバー）
6. 線を **100%** なぞりきったらクリア（100%以外の完了条件はありません）

一筆書きの必要はなく、どこからでも好きな順番でなぞれます。
針先が線に近づくと軽く吸い付く補正が入っていて、多少手が震えても大きくズレにくくなっています。
クリアタイムが画面に表示されます（ランキング機能は未実装）。

## 動かし方

`index.html` をブラウザで開くだけで動きます。ビルドやサーバーは不要です。
スマホでの動作確認を推奨します（`touchstart`/`touchmove` を使用しているため、PCではマウスドラッグで代用できます）。

```
open index.html
```

またはローカルサーバーで:

```
npx serve .
```

GitHub Pages等の静的ホスティングにそのまま配置しても動作します。

## ゲームの仕組み（概要）

- 各ステージの輪郭は「中心からの角度ごとの半径」を返す関数（`script.js` の `STAGES` 配列）として表現しています。
  円だけでなく、風鈴・うちわ・金魚・提灯のような複雑な形も同じ判定ロジックで扱えます。
  局所的な特徴（持ち手・結び目・尾びれ・羽根など）は `bump()` という滑らかな山型関数を輪郭に足し引きして作っています。
- 輪郭を360分割した「バケツ」に分け、緑判定で通過したバケツを記録して進捗（トレース率）を出しています。クリアには全360バケツの通過が必要です（100%固定、緩和条件はありません）。
- 指の位置と実際の判定点（針先）をあえてズラすことで、指で判定点が隠れないようにしています。針は指から少し浮いた位置に描画されます。
- 針先が線に近づくと、判定位置がわずかに線へ吸い寄せられる補正が入っています（自動操作にはならない範囲の軽い補正）。
- 状態が変化した瞬間（緑・黄・赤）に振動（Vibration API）を鳴らし、緑でなぞっている間は削り音と軽い振動が断続的に鳴ります。
  - iOS Safari は Vibration API 非対応のため振動しません（ブラウザ側の制限）。
  - Android Chrome 等では動作します。
- 失敗すると型が破片になって飛び散る演出、クリアすると一拍静止したあと型がふわっと持ち上がり、下に穴と影が残る演出が入っています（日の丸ステージのみ、持ち上がると背後に白い旗地が見えます）。
